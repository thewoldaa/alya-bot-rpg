# Setup

1. Install Node.js versi modern.
2. Jalankan:
   ```bash
   npm install
   ```
3. Jalankan bot:
   ```bash
   node index.js
   ```
4. Saat `.env` belum ada, bot akan meminta:
   `Masukkan TOKEN BOT:`
   lalu menyimpannya otomatis ke `.env`.

## Environment

Default sudah cukup untuk jalan:

- `TOKEN`
- `PREFIX` opsional, default `.`
- `OWNER_IDS` opsional, default `111218344834482`
- `OWNER_NAME` opsional, default `Craftkal`
- `GUILD_ID` opsional untuk registrasi slash command di server tertentu
- `DEFAULT_USER_LIMIT` opsional, default `10`
