# Aturan Bot

1. Semua user wajib `register` atau `login` dulu sebelum memakai fitur utama.
2. `backupCode` bersifat rahasia, tapi bisa dipakai berkali-kali untuk akun lain yang kamu miliki.
3. Satu `core_id` berarti satu data shared. Jangan membuat data ganda untuk akun yang sama.
4. Command owner seperti `.addmoney` dan `.buatredeem` hanya untuk owner yang ditentukan di config.
5. Anti-spam aktif otomatis. Jika mengirim terlalu cepat, akun akan diblokir sementara.
6. Gunakan command ekonomi dengan bijak. Saldo tidak bisa minus.
7. Data pasangan, anak, rumah, inventory, dan portfolio disimpan permanen di database lokal.
