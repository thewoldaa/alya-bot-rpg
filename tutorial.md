# Tutorial

## 1. Start pertama

1. Jalankan `node index.js`.
2. Masukkan token saat diminta.
3. Pastikan bot online di Discord.

## 2. Register

```txt
.register NamaKamu
```

Bot akan membuat `core_id` dan mengirim `backupCode` ke DM.

## 3. Login akun lain

```txt
/login id
```

Akun lain akan terhubung ke data yang sama, jadi uang, inventory, pasangan, dan stock portfolio ikut shared.

## 4. Fitur utama

- Ekonomi: `.work`, `.shop`, `.inv`, `.pay`, `.slot`, `.togel`
- Job dan makan: `.job`, `.mk`, `.work`
- RPG Gear: `.gear`, `.equip`, `.unequip`, `.enchant`
- Sosial: `.belirumah`, `.lamar`, `.pd 1-4`, `.anak`, `.nkkand`, `.afk`
- Aksi sosial: tombol `Act`, `Beri Makan`, dan `Beri Uang` ada di `.pd`
- Lainnya: `.character`, `.ind`, `.redeem`, `.menu`, `.help`

## 5. Slash command server

- `/setchat`
- `/setuserlimit`
- `/resetserver`
